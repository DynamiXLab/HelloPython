# 第1章 Python 是什么 & 安装 Thonny

> **难度**：⭐  
> **所属教程**：Python 零基础入门到 MicroPython 嵌入式实战

---

### 1.1 Python 是什么

Python 是一门**解释型**编程语言，语法接近人类语言（英语），非常适合初学者。它可以用来做网站、数据分析、人工智能，也能在指甲盖大小的芯片上运行——这就是我们后半部分要玩的 MicroPython。

### 1.2 选一个编辑器——Thonny 还是 VS Code？

本教程前半部分（第1~14章，纯 Python）用任何 Python 编辑器都可以。有两个推荐：

**Thonny**（初学者首选，也是本教程后半部分必需的）

- 安装简单，自带 Python，不需要单独配置环境
- 变量值可视化显示
- 支持 MicroPython，可以直接给 ESP32 写代码（第15章起必需）
- 界面清爽，菜单不吓人

**VS Code**（如果你已经装了，或想用一个更通用的编辑器）

- 免费开源，功能强大
- 需要单独安装 Python（从 python.org 下载），但配置一次就行
- 插件丰富，写代码体验很好
- 注意：**第15章起必须换回 Thonny**，因为 VS Code 不支持直接用 MicroPython 给 ESP32 烧录代码

> 无论选哪个，前半部分的 Python 代码都是完全通用的。选你更顺手的就行——Thonny 更省心（不用配环境），VS Code 更强大（如果你以后想继续写代码）。

### 1.3 安装 Thonny

1. 打开浏览器访问 **https://thonny.org**
2. 点击首页大大的下载按钮，选择对应你的系统版本（Windows / macOS / Linux）
3. 下载完成后双击安装，一路点"下一步"即可（全部用默认设置）
4. 安装完成后打开 Thonny，你会看到一个白色编辑区和下方的 Shell 窗口

> **验证安装**：在 Thonny 下方的 Shell 区域输入 `print("hello")` 然后按回车，看到输出 `hello` 就说明安装成功了。

### 练习任务

**任务（⭐ 容易）**：成功安装 Thonny，并在 Shell 中输入以下内容然后按回车：

```python
print("我是XXX，今天开始学Python！")
```

把 `XXX` 替换成你的名字。截图保存你的第一个 Python 输出，这是你编程之旅的起点！

**任务（⭐⭐ 中等）**：在 Shell 中尝试以下操作，观察输出结果并记录：

```python
print(3 + 5)
print("hello" + "world")
print("hello" * 3)
```

思考：`+` 用在数字和用在字符串上，行为有什么不同？

**任务（⭐⭐⭐ 困难）**：尝试只用 `print()` 语句（可以用多条），输出以下图案：

```
*
**
***
****
*****
```

> 提示：字符串可以用 `*` 重复，比如 `"*" * 3` 的结果是 `"***"`。

💡 **思考延续**

"Python"这个名字和蟒蛇没关系。1989年圣诞节假期，荷兰程序员 **Guido van Rossum** 在阿姆斯特丹的家里闲着无聊，决定写一门新语言打发时间——他当时刚读完一本关于 ABC 语言（一个教学用编程语言）的报告，觉得有很多可以改进的地方。

至于名字，他当时正在看 BBC 一部叫《Monty Python's Flying Circus》的喜剧。他觉得"Python"这个词有种"神秘又不失幽默"的感觉，正好和这门语言"写起来有意思"的风格吻合。

后来 Python 大火，大家习惯用两条蛇缠绕的 🐍 做标志。但如果你问 Guido 本人，他会告诉你：这头蛇不是丛林里的大蟒蛇，是被蒙提·派森逗笑的那条。
