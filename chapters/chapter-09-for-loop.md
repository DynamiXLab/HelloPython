### 9.1 for 循环基础

```python
# 打印 0 到 4
for i in range(5):
    print(i)

# 输出：
# 0
# 1
# 2
# 3
# 4
```

`range(5)` 生成 0, 1, 2, 3, 4 一共 5 个数字。**和索引一样，从 0 开始，不包括 5**。

### 9.2 range() 的三种用法

```python
range(5)          # 0, 1, 2, 3, 4
range(1, 5)       # 1, 2, 3, 4（从1开始）
range(1, 10, 2)   # 1, 3, 5, 7, 9（步长为2）
```

### 9.3 遍历列表

```python
fruits = ["苹果", "香蕉", "橘子"]
for fruit in fruits:
    print(f"我喜欢吃{fruit}")
# 迭代变量 fruit 依次取列表中的每个元素
```

### 9.4 累加器模式

```python
# 计算 1 到 100 的和
total = 0
for i in range(1, 101):
    total = total + i    # 也可以用 total += i
print(total)  # 5050
```

### 练习任务

**任务（⭐ 容易）**：用 for 循环打印 1 到 10 的乘法表。给定一个数字 `n`（比如 `n = 7`），输出：

```
7 × 1 = 7
7 × 2 = 14
...
7 × 10 = 70
```

**任务（⭐⭐ 中等）**：用 for 循环输出**9×9 乘法表**（阶梯形），格式如下：

```
1×1=1
2×1=2  2×2=4
3×1=3  3×2=6  3×3=9
...
9×1=9  9×2=18  ...  9×9=81
```

> 提示：外层循环 `for i in range(1, 10)`，内层循环 `for j in range(1, i+1)`，每行用 `print(..., end="	")`，行末用空 `print()` 换行。

**任务（⭐⭐⭐ 困难）**：输出一个"阶梯乘法表"，但只保留乘积为**偶数**的结果：

```
1×2=2
2×1=2  2×2=4  2×3=6  2×4=8
3×2=6  3×4=12
...
9×2=18  9×4=36  9×6=54  9×8=72
```

> 提示：在内层循环中用 `if (i * j) % 2 == 0` 判断乘积是否为偶数。综合运用：for 循环（第9章）+ 条件判断（第7章）+ `range()` 步长。

💡 **思考延续**

`for` 循环这个概念，比第一台电子计算机早了整整一个世纪。

1843 年，英国数学家 **Ada Lovelace** 在翻译一篇关于 Charles Babbage "分析机"（一种纯机械的、由齿轮和曲柄驱动的计算设备）的法语论文时，加入了大量自己的注释。在这些注释中，她写下了一段详细的算法——如何让分析机**反复执行同一组操作来计算伯努利数**。后世认为这就是世界上第一个计算机程序。

Ada 比所有人都早意识到一件事：机器不只可以算数字，它还可以**重复执行一组指令**——只要记录下上一轮的结果，用它作为下一轮的输入。这就是循环的核心思想。她还预见到这种机器可以用来处理音乐符号、文字和其他非数值信息，这在一百多年后才变成现实。

所以当你写 `for i in range(5):` 的时候，你踩在一条绵延 180 年的思考传统上。Ada Lovelace 在纸上写下那份笔记时，大概想象不到，一个多世纪后的初学者每天都可以在几毫秒内完成她当年只能靠想象的计算。
