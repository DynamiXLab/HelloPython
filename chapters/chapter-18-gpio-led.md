### 18.1 什么是 GPIO

GPIO（General Purpose Input/Output）即**通用输入输出引脚**。你可以通过代码控制这些引脚输出高电平（3.3V）或低电平（0V）。

### 18.2 硬件连接

```
ESP32 引脚 D13(GPIO13) ──── 220Ω 电阻 ──── LED 正极（长脚）
                                            │
                                         LED 负极（短脚） ──── GND
```

> **关键**：LED 长脚接正极，短脚接 GND。**电阻必须串接，否则 LED 会烧掉**。

### 18.3 电阻怎么选？——用欧姆定律算一遍

你可能好奇：为什么是 220Ω？100Ω 不行吗？不接电阻会怎样？

**先认识 LED 的两个参数：**

- LED 点亮时需要约 2V 的**正向压降**（不同颜色略有差异：红色约 1.8~2.2V，蓝色约 3.0~3.2V）
- LED 正常工作需要的电流约 5~20mA（毫安）。电流越大 LED 越亮，但超过上限会烧坏

**用欧姆定律算电阻值：**

欧姆定律：`U = I × R`，也就是 `R = U ÷ I`

在这个电路里：
- ESP32 引脚输出电压 = 3.3V
- LED 吃掉 2V
- 剩下的电压（3.3V - 2V = 1.3V）落在电阻上
- 我们想让电流控制在 10mA = 0.01A（安全亮度）

```
R = (3.3V - 2V) ÷ 0.01A = 1.3 ÷ 0.01 = 130Ω
```

为什么实际用 220Ω 而不是 130Ω？两个原因：

1. **留安全余量**：LED 实际压降可能有偏差，电阻用大一点只会让 LED 稍微暗一点，但用小了可能烧坏
2. **常见电阻值**：标准电阻序列里有 220Ω，容易买到；130Ω 不在标准序列里

> 你可以试试：把 220Ω 换成 330Ω 或 470Ω，LED 会变暗但依然能亮。把 220Ω 换成 100Ω，LED 会更亮，但更接近 ESP32 引脚的电流上限。

**为什么不接电阻会烧？**

没有电阻时电路是这样的：GPIO 输出 3.3V → LED（压降 2V）→ 直接连 GND。剩下的 1.3V 出现在导线上——导线的电阻几乎为 0Ω。欧姆定律告诉你：

```
I = U ÷ R = 1.3V ÷ 0Ω → 电流趋近无穷大
```

实际上 LED 和 ESP32 引脚会先烧掉，电路断开后再也没有电流。**串联电阻的作用就是"限制电流"**——用一段安全的电阻把电流压到 LED 和引脚都能承受的范围。

### 18.4 ESP32 引脚电流限制

ESP32 每个 GPIO 引脚的电流不是无限供应：

- **单个引脚最大**：约 12mA（默认配置），可软件调整上限到约 40mA
- **安全建议**：新手控制在 **≤10mA 每引脚**，给硬件留冗余
- **所有引脚总电流**：受板载稳压器限制，通常不超过 500mA

> 驱动单个 LED 用 220Ω 电阻绰绰有余。如果以后要驱动大功率设备（电机、大功率灯泡），不要直接从 GPIO 取电——加一个**三极管**或**MOS 管**来"放大"电流。

### 18.5 代码——点亮 LED

```python
from machine import Pin
import time

# GPIO13 设为输出模式
led = Pin(13, Pin.OUT)

# 点亮 LED
led.value(1)     # 1 = 高电平 = 亮
time.sleep(2)    # 保持 2 秒
led.value(0)     # 0 = 低电平 = 灭
```

### 18.6 LED 闪烁

```python
from machine import Pin
import time

led = Pin(13, Pin.OUT)

while True:
    led.value(1)     # 亮
    time.sleep(0.5)  # 等 0.5 秒
    led.value(0)     # 灭
    time.sleep(0.5)  # 等 0.5 秒
```

你会看到 LED 以每秒一次的频率闪烁。恭喜，你写的代码已经能影响物理世界了！

```
电路参考：
GPIO13 ──[220Ω]── LED1 ── GND
GPIO12 ──[220Ω]── LED2 ── GND
GPIO14 ──[220Ω]── LED3 ── GND
```

> 提示：定义列表 `leds = [led1, led2, led3]`，用 for 循环控制。

**任务（⭐⭐⭐ 困难）**：用 3 个 LED 实现**倒计时灯**：
