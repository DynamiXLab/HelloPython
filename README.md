# HelloPython — Python 零基础到 MicroPython 嵌入式实战

> 🎯 从零开始学 Python，后半段用 ESP32 做嵌入式项目。22 章，每章有练习。

## 📚 教程结构

| 阶段 | 章节 | 内容 |
|------|------|------|
| **Python 基础** | 第1~5章 | 环境搭建、变量、数字、字符串 |
| **交互与逻辑** | 第6~10章 | 用户输入、条件判断、列表、循环 |
| **数据结构与函数** | 第11~12章 | 字典与元组、函数 |
| **面向对象** | 第13、16章 | OOP 入门（简单）→ OOP 进阶（强化提高） |
| **工程化** | 第14~15章 | 文件读写、异常处理 |
| **嵌入式实战** | 第17~22章 | ESP32 + MicroPython：LED、按键、PWM、传感器、OLED |

## 🛠 所需工具

- **编辑器**：Thonny（https://thonny.org）
- **硬件**（第17章起）：ESP32 开发板 + 基础元件套件（约 60~80 元）

## 📖 使用方式

从头按章节顺序阅读，每章末尾有练习任务（简单 + 挑战两个级别）。所有章节文件在 [`chapters/`](chapters/) 目录下。
