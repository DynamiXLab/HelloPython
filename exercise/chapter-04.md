# 第4章 参考答案

## ⭐ 容易

```python
r = 5
area = 3.14 * r ** 2
print(f"圆的面积：{area}")  # 78.5
```

## ⭐⭐ 中等

```python
num = 357
bai = num // 100
shi = (num // 10) % 10
ge = num % 10
print(f"百位：{bai}")
print(f"十位：{shi}")
print(f"个位：{ge}")
```

## ⭐⭐⭐ 困难

```python
distance = 3.6
minutes = 45
hours = minutes / 60
speed = distance / hours
print("距离：3.6 公里")
print("时间：45 分钟")
print(f"平均速度：{speed:.1f} 公里/小时")
```
