# 第1章 参考答案

## ⭐ 容易

安装 Thonny，在 Shell 中输入：

```python
print("我是XXX，今天开始学Python！")
```

把 `XXX` 替换成你的名字，截图保存。

## ⭐⭐ 中等

```python
print(3 + 5)          # 8（数字加法）
print("hello" + "world")  # "helloworld"（字符串拼接）
print("hello" * 3)    # "hellohellohello"（字符串重复）
```

`+` 用在数字上做数学加法，用在字符串上做拼接。
