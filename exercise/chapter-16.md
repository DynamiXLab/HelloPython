# 第16章 参考答案

## ⭐ 容易（阅读题）

1. 嵌入式系统是藏在设备内部的小型计算机，只完成一两个特定任务；普通电脑是通用计算机，运行多种软件。
2. ESP32 优势：双核 240MHz、内置 Wi-Fi 蓝牙、价格便宜、支持 MicroPython、GPIO 丰富。

## ⭐⭐ 中等

搜索"ESP32 DevKit V1 pinout"，记录引脚位置：GPIO13、12、14（LED）、GPIO25（按键）、GPIO21/22（I2C）、3.3V/GND/5V。

## ⭐⭐⭐ 困难

| 维度 | ESP32 | STM32F103 |
|------|-------|-----------|
| 主频 | 240MHz 双核 | 72MHz 单核 |
| 内存 | ~520KB SRAM | ~20~96KB SRAM |
| WiFi/蓝牙 | ✅ 内置 | ❌ 无 |
| 价格 | ~20~30元 | ~10~20元 |
| 编程 | MicroPython/C | C/C++ |
| 场景 | IoT、智能家居 | 工控、电机控制 |
